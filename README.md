# Projet 9 - Big Data sur le Cloud

Base du projet OpenClassrooms P9.

Objectif : construire un pipeline PySpark sur AWS pour traiter des images de fruits, extraire des features avec un modele CNN pre-entraine, reduire les dimensions avec une PCA et sauvegarder les resultats.

Structure prevue :

- notebooks : notebooks d'exploration et pipeline principal.
- src : fonctions Python reutilisables.
- docs : notes AWS, RGPD et soutenance.
- data : dossier local ignore par Git.

Le code sera commente en francais avec des commentaires simples et des explications Markdown pertinentes.
